ÍNDEX

  1. Com puc cambiar el tamany de la lletra?

    Per poder cambiar-li el tamany has de seleccionar el text que vulguis augmentar o disminuir, i despres hauras d'elegir el tamany prement en els nombres (Que devora tenen un + i un -) de la barra de dalt.

  2. Com puc posar la lletra en negrta?

    Per poder cambiar-li la lletra en negrta, pots fer varies coses: - Seleccionar el text i premer Ctrl + b. - Seleccionar el text i premer la lletra 'B' de la barra de dalt

  3. Com puc fer perque el text estigui centrat?

    Per poder fer que el text estigui centrat pots fer varies coses: - Seleccionar el text i premer Ctrl + Mayús + E. - Seleccionar el text, premer, l'icona de les 5 retxes (De la barra de dalt), i elegir el que les retxes estan centrades.

  4. Com puc fer perque el text estigui centrat?

    Per poder fer que el text estigui centrat pots fer varies coses: - Seleccionar el text i premer Ctrl + Mayús + E. - Seleccionar el text, premer, l'icona de les 5 retxes (De la barra de dalt), i elegir el que les retxes estan centrades.

  5. Com puc fer per cambiar el color del text?

    Per poder cambiar el color del text, has de premer l'icona de la 'A' amb un retangle devall (Dels color sleccionat), una vegade hagis clicat ja podrás elegir el color que vulguis.

  6. Com puc fer per descarregarme un arxiu PDF?

    Per poder descarregarte un arxiu PDF, has de anar a Fitxer, després a Baixades, i allà dins seleccionar el tipus que vulguis, en el nostre cas Document PDF.

  7. Com puc afegir una imatge?

    Per poder afegir una imatge, ha de premer a Insereix, i allà tindras varies opcions, hauras d'elegir Imatge, despres se te desplegara un menú, on hauras d'elegir desde on vols penjar l'imatge.

  8. Com puc compartir el document?

    Per poder compartir el document ho pots fer de varies maneres: - Prement l'icona de Comparteix, i després posant els gmails de les persones que vols afegir. - Prement Fixer, després premer comparteix, allà se te desplegara un menú on haurem de seleccionar l'opció de Comparteix amb altres persones, i per últim hauras de posar els gmails de les persones que vols afegir.

  9. Com puc afegir un enllaç

    Per poder afegir un enllaç, ho pots fer de varies maneres: - Seleccionar el text que vols que dugui l'enllaç, premer Ctrl + K, i afegir l'enllaç. - Seleccionar el text que vols que dugui l'enllaç, i premer l'icona de (mig cercle)(-)(Mig cercle), una vegada se te hagi obert, només faltara afegir l'enllaç.

  10. Com puc afegir nombres de pàgina?

    Per poder afegir els nombres, has d'anar a Insereix, i si baixas sortirà l'opció de Nombres de pàgina, una vegada hagis clicat, et sortiran varies opcions, elegeixes la que mes t'agradi.
